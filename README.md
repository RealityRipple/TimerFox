# TimerFox
Easily counts down from a predefined number and alarms you when the count has reached zero.

#### Supports
 * SeaMonkey [2.0a1 - 2.*]

## Building
Simply download the contents of the repository and pack the contents (sans git data) into a .zip file. Then, rename the file to .xpi and drag into the browser.

## Download
You can grab the latest release from the [Official Web Site](//addons.thunderbird.net/en-US/seamonkey/addon/timerfox/).
